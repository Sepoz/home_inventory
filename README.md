## Laravel template 1.0

-   $ git clone git@gitlab.com:hackademy33/laravel_template.git
-   $ cd laravel_template
-   $ rm -rf .git

-   Crea un nuovo repository
-   Segui le indicazioni di per collegarlo a laravel_template

-   $ composer install
-   $ cp .env.example .env
-   $ php artisan key:generate

-   $ npm install

-   $ npm run dev
